# Java Mini Projects

This repository contains beginner-friendly Java projects developed using core Java concepts. These projects helped me improve my problem-solving skills, object-oriented programming knowledge, and understanding of Java fundamentals.

---

## Projects Included

### 1. ATM Interface
A simple ATM simulation project that allows users to:
- Check account balance
- Deposit money
- Withdraw money
- View transaction messages

#### Concepts Used
- Classes & Objects
- Methods
- Conditional Statements
- User Input using Scanner

---

### 2. Currency Converter
A Java application used to convert one currency into another using exchange rates entered by the user.

#### Features
- Select base currency
- Select target currency
- Enter amount
- Currency conversion
- Display converted amount

#### Concepts Used
- Variables & Data Types
- Arithmetic Operations
- User Input
- Simple Calculations

---

### 3. Student Management System
A basic system to manage student details.

#### Features
- Add student details
- View student records
- Store student information
- Simple management operations

#### Concepts Used
- Arrays / ArrayList
- Classes & Objects
- Loops
- Methods
